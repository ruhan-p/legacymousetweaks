# Legacy Mouse Tweaks

A mod that enhances inventory management by adding various functions to the mouse buttons. A 1.8.9 Port of YaLTeR's original mod.

# Original Mod

## For more details, visit the original Mouse Tweaks modrinth page:
https://modrinth.com/mod/mouse-tweaks

## And the original repo:
https://github.com/YaLTeR/MouseTweaks

# How to build

`gradlew build`

# Compatibility

Mouse Tweaks should work with everything based on `AbstractContainerScreen`. If your GUI isn't based
on `AbstractContainerScreen`, or if you want to provide additional compatibility, take a look
at `src/main/java/yalter/mousetweaks/api/`.
